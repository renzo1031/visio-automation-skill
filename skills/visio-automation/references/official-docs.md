# Official Docs

Use this file only when a Visio COM, VBA, or ShapeSheet detail is unclear.

## Rule

Prefer Microsoft Learn for Visio COM/VBA and ShapeSheet behavior. If the official docs are unclear, treat third-party examples only as hints and verify the behavior with a tiny local Visio script before relying on it.

Search by the exact object, method, or ShapeSheet cell name, such as `Documents.OpenEx`, `ShapeRouteStyle`, `ConLineRouteExt`, `GlueToPos`, or `CellExistsU`.
